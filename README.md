## Bonus: Getting 10x more from every prompt

- **Stack context.** Start any session with: "You're an expert UK estate agency coach. My agency: [NAME], area: [AREA], my role: [ROLE]." Every answer after that gets sharper.
- **Ask for variants.** Add "give me 3 versions: safe, bold, and unexpected" to any writing prompt.
- **Feed it your voice.** Paste 2-3 things you've written and add: "match this tone in everything you write for me."
- **Make it critique.** After any draft: "Now be a sceptical homeowner reading this. What don't you believe?"
- **Never paste client personal data.** Anonymise names, addresses, and financial details before using any AI tool — treat compliance as non-negotiable.
